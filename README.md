# Tkinter Calculator

## Description

This is a simple Calculator application developed using Python and the Tkinter GUI library. The calculator performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division. It also provides features like Clear (AC) and Backspace.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (×)
* Division (÷)
* Clear all input (AC)
* Delete last digit (Backspace)
* Supports 0, 00, and 000 buttons
* Error handling for division by zero

## Requirements

* Python 3.x
* Tkinter (comes pre-installed with Python)

## How to Run

1. Install Python 3 if not already installed.
2. Save the calculator code in a file named `calculator.py`.
3. Open Command Prompt or Terminal.
4. Navigate to the project folder.
5. Run the following command:

```bash
python calculator.py
```


## Output

The application opens a graphical calculator window where users can:

* Enter numbers using buttons.
* Perform arithmetic operations.
* View results in the display box.
